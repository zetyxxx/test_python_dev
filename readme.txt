1. Устанавливаем необходимые библиотеки из requirements.txt следующей командой:
pip install -r requirements.txt
2. В файле config.py прописываем параметры коннекта к БД (host, user, password, имя базы данных)
3. Далее запускаем файл test.py для проведения прописанных тестов

В main.py расположена основная функция polinom, выделенная абстракция write_to_db, которая записывает дату, название функции,
исходные аргументы и результат либо ошибку в базу данных postgreSQL

В test.py прописаны 5 юнит-тестов, 1 с большим объемом входных данных, 3 с неправильным форматом входных данных и 1 корректный,
с входными данными прекрасно описывающими полином 3-ей степени.
Ошибки обрабатываются корректно и записываются в бд.
----------------------------------------------------------------------------------------------
- Чтбы развернуть проект через докер:
Перед началом необходимо поменять в файле config.py значение переменной host на host="db", а также в файле docker-compose.yml
поменять значения POSTGRES_DB, POSTGRES_USER, POSTGRES_PASSWORD на такие же как выставили в файле конфига
Далее вводим команды в терминале:
docker-compose build
docker-compose up (с первого раза может не получится, поэтому ctrl-C и по новой docker-compose up)
