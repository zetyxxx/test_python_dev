1. Устанавливаем необходимые библиотеки из requirements.txt следующей командой:
pip install -r requirements.txt
2. В файле config.py прописываем параметры коннекта
3. Далее запускаем файл db_create.py чтобы создать таблицу в бд

В main.py расположена основная функция polinom, выделенная абстракция write_to_db, которая записывает дату, название функции,
исходные аргументы и результат либо ошибку

В test.py прописаны 4 юнит-теста, 1 с большим объемом входных данных, другие с неправильным форматом входных данных.
Ошибка обрабатывается корректно и записывается в бд.
